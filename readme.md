# a promise queue

This is just another promise queue. Simple.

+ Native es6
+ No concurency
+ Optional retry attempts for failed promises
+ Option to use your faviour promise flavour (Bluebird, Q)

## Install

You know this:
```
npm install a-promise-queue
```
